XD-1 is written by Pierre Sprimont in Bologna, Italy, at the national astrophysic institute INAF/IASF. The source-code is available online in the following Github repositories :
  • XD-1 : https://github.com/Nunkiii/XD-1
  • Qk/sadira : https://github.com/Nunkiii/sadira

Why the name XD-1 ?

This is the code-name of the spaceship Discovery One appearing in the film 2001: A Space Odissey.

Dependencies

This viewer makes use of the (still buggy) astro.js fits.js JavaScript FITS library for local FITS file loading in the browser.
